# DNS AAAA record

An **AAAA record** (quad-A record) maps a domain name to an **IPv6 address** (e.g., `example.com → 2606:2800:220:1:248:1893:25c8:1946`). It is the IPv6 equivalent of the A record. As IPv4 addresses become exhausted, AAAA records are increasingly essential for modern internet connectivity.
