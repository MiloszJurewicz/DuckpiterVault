# DNS A record

An **A record** (Address record) maps a domain name to an **IPv4 address** (e.g., `example.com → 93.184.216.34`). It is the most fundamental DNS record type — when you type a domain into a browser, the A record is what resolves it to the server's IP. A single domain can have multiple A records for load balancing (DNS round-robin).
